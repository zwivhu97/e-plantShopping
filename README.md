# e-plantShopping 🌿

## Project Overview

**e-plantShopping** is a React and Redux based web application that allows users to browse plants and add them to a shopping cart. The application simulates an online plant store where customers can select plants, manage their cart, and view the total cost of items before checkout.

## Features

* Browse different plants available for purchase
* Add plants to the shopping cart
* Increase or decrease the quantity of items in the cart
* Remove items from the cart
* View the total cart amount
* Continue shopping or proceed to checkout

## Technologies Used

* React
* Redux Toolkit
* JavaScript
* CSS

## How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies

```
npm install
```

4. Start the application

```
npm run preview
```

The application will run on:

```
http://localhost:4173
```
